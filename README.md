# csharp

## FileDateTimeEditor
Программа для редактирования времени создания, доступа, измения файлов.
Windows Forms.

## ForbiddenWords
Программа находит все файлы с расширение .txt и проверяет их на наличие указанных пользователем запрещенных слов. 
Программа заменяет слова символом * и копирует оригинальные файлы, а также создает новые с измененным текстом в директорию, указанную пользователем.
Winodws Forms, многопоточность, progress bar.

## RemoteFileAccess
Практика с сетью, сокетами.
Программа позволяет заходить в папки, просматривать файлы, копировать в директорию, в которой находится .exe файл.
Имена файлов, папок, копирование происходит через TcpListener.
Console app

## BookStore
Практика с Dapper, базой данных MS SQL, написание запросов.
Создание авторов, книг, отображение, просмотр статистики.
Windows Forms.

## DbManagerEntityFramework
Практика с EntityFramework.
Добавление/удаление отделов, работников.
Windows Forms.

## SomeWebApplication
Практика с ASP.NET MVC Framework, EntityFramework.
Добавление книг, просмотр, поиск, добавление комментариев, пагинация.
